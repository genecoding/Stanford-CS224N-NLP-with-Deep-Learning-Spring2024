# Stanford CS224N Natural Language Processing with Deep Learning
My implementation for the assignments from the course of Stanford, CS224N Natural Language Processing with Deep Learning, Spring 2024.

## Information
* Instructor: Chris Manning

## Contents
* Assignment 1: Introduction to word vectors
* Assignment 2 (Winter 2024): Derivatives and implementation of word2vec algorithm
* Assignment 2: Dependency parsing and neural network foundations
* Assignment 3: Neural Machine Translation with sequence-to-sequence, attention, and subwords
* Assignment 4: Self-supervised learning and fine-tuning with Transformers
* Final Project: Default Final Project
